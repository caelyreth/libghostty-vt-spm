### GhosttyVtSPM

Minimal Swift Package wrapper for upstream `libghostty-vt`.

### Usage

Add this package in SwiftPM and import the public module:

```swift
import GhosttyVt
```

The package exposes the C API from Ghostty's `include/ghostty/vt.h`.

### Platforms

Minimal supported: macOS 14, iOS 17, and macCatalyst 17.
